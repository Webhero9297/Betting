This is hybrid sport bets application based on EVM blockchain technology.


Games List

All of the data returned from classic backend service. Only bets processed by metamask plugin. Bet message include next data fields: gameId, teamId, bet amount.
After game match ends, backend do calculation and sends message with payments to the ethereum network.

My bets

Page that contains all bets that you sent to the ethereum network.

Since EVM has no possibility to calc decimal, currently backend service do calculations.


RULES:

- When game canceled, sums refunded as is, without any changes.
- When 1 team wins, win and lose sums would depend on game scores, as much difference, win and lost sums would be more significant, including case when score 0:1 then lose bets would receive 0, because in percentage the difference are 100%.
- When draw score won, then from each bet that did not put on draw, withdraw -(100%/amount of teams(usually it is 2)) of total bet sum
- When draw score lost, then draw bets would receive -50%(same as described upper) of total bet sum
Information how to setup metamask plugin you can find at google.com
